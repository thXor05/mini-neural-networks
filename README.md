# mini-neural-networks

A relatively small and simple implementation of some basic neural network architectures, written entirely in Python.
This implementation does not rely on any external libraries such as NumPy or PyTorch, but uses only some of Python's standard libraries in order to provide a deeper understanding of the architectures being implemented.
Each implementation is kept as simple, compact and general as possible and can be used and extended for educational purposes or small projects.
Feel free to use the entire code for whatever you want!

## Project Overview

### models

Contains an implementation of each individual model as a Python class.
In most cases, the file can be directly added to a project and used immediately.

### training

Contains sample training loops for each specific architecture that can be used for experiments or extended for specific applications.

### utils

Contains utils for mathematical calculations.
